## Install SQLite3 for Linux/Ubuntu

```
sudo apt update
sudo apt install sqlite3
sqlite3 --version
```

## Replace Disc Token

Go to ```discord.php``` then replace the token

```
const DISCORD_TOKEN = 'YOUR_TOKEN_HERE';
```

## Usage

```
php sqlite.php
php discord.php
```
